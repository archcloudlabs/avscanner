## AVScanner
A silly java app for students to explore stdin, File I/O, and unit tests.


## Building & Running
```
$> mvn clean compile
$> mvn package
$> java -cp ./target/avscanner-1.0.jar com.archcloudlabs.App

Enter file path to scan: 
./Data/data.txt
EVIL BYTES
WARNING FOUND MALICOUS FILE: data.txt
```
